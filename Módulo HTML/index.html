O que é HTML

HTML significa Linguagem de Marcação de HiperTexto ou em inglês HiperText Markup Language. Esta linguagem foi desenvolvida para definir a estrutura de um conteúdo de uma página da web através de separadores denominados tags.

É necesssário convencionarmos aqui, que apesar de ser uma linguagem, o HTML não é própriamente uma linguagem de programação, pois o mesmo não é capaz de criar funcionalidades dinâmicas.

No máximo que podemos argumentar é que pode-se desenvolver páginas em HTML, mas nunca deve-se dizer que podemos programar em HTML.

Tags

Tags são elementos encontrado no HTML que tem por objetivo aplicar distntas formatções aos conteúdos das páginas web de acordo com a tag utilizada, no HTML como algumas das principais tags podemos citar:

  • Tag <head> - Tag de cabeçalho visível na aba do navegador;
  • Tag <body> - Tag de corpo da página é dentro desta tag que todo conteúdo será exibido na página;
  • Tag <h1 a h6> - Tag que defini tamanhos dos conteúdos (Títulos e Subtítulos);
  • Tag <img> - Tag para inserir mídia, esta necessita o atributo src para anexar a mídia corretamente;
  • Tag <a> - Tag âncora necessária para vincular uma URL ao conteúdo, necessário atributo href para direcionar o usuário;
  • Tag <p> - Tag para iniciar um paragráfo de texto simples;
  • Tag <ol> - Tag que cria listas ordenadas, necessita conter a tag <li> em seu interior para que as lista sejam criadas corretamente;
  • Tag <ul> - Tag que cria listas não-ordenadas, necessita conter a tag <li> em seu interior para que as lista sejam criadas corretamente;
  • Tag <il> - Tag que cria itens dentro de listas;
  • Tag <mark> - Tag que destaca palavras ou trechos de texto similar a uma caneta marca-texto;
  • Tag <blockquote> - Tag para incluir citações de obras e/ou autores.

Atributos

São complementos de tags encontrados dentro destas ultimas reponsáveis por atribuir certas configurações a algumas tags.

Por exemplo a tag <a> necessita do atributo href para que este ultimo aponte a URL para uma página web, ou a tag <img> demande o atributo src para apontar o endereço de uma mídia local ou remota.
